Welcome to the Nextflow Workspace

This is your personal workspace. The "pd" folder represents your persistent drive:

  • The files you save here will still be available when you come back after terminating your workspace session.
  • Any personal files outside of this folder will be lost.

Get started with Nextflow

If you are new to Nextflow, visit nextflow.io for detailed information.

This workspace is set up to run Nextflow workflows in AWS Batch. Your Nextflow configuration must include the Batch queue, IAM role ARN and work directory that were created for you automatically. The sample configuration file will allow you to run simple workflows and can be adapted to your needs. To get started:

  • Copy the contents of your sample configuration file to a new "nextflow.config" file.
  • Open a new "Launcher" tab and open a terminal.
  • Run the Hello World workflow in the terminal with command "nextflow run hello".
